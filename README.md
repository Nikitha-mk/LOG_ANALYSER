# 📝 Log Analyzer Tool (Python)

## 📌 Description
This is a simple Python-based Log Analyzer Tool that reads system log files and categorizes log entries into:
- Error
- Warning
- Info

It also counts occurrences of each type to help in debugging and system monitoring.

## 🚀 Features
- Parses log files
- Identifies log levels (Error, Warning, Info)
- Counts occurrences
- Beginner-friendly implementation

## 🛠️ Technologies Used
- Python

2. Open terminal in project folder
3. Run
